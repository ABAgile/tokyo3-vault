Machine tokens are bearer credentials for automated callers (CI pipelines, scripts, service accounts). They authenticate against the JSON API with Authorization: Bearer <token> and inherit the issuing user's project access — unless restricted by a scope below. Read-only tokens block all write operations regardless of scope.
